6.3 CSS 클래스 바인딩

  • CSS 클래스를 지정하기 위해 v-bind:class="..." 또는 :class="..." 사용
  • 2가지 방법:
    1. CSS 클래스명 문자열을 바인딩하는 방법
    2. true/false 값을 가진 객체를 바인딩하는 방법

1) CSS 클래스명 문자열을 바인딩하는 방법

  • 데이터나 속성에 CSS 클래스명 문자열을 할당하여 이 값을 :class로 바인딩하는 방법
  • 여러 개의 데이터 값을 클래스로 바인딩 하는 방법
  • 3항 연산식과 조합하면 동적으로 클래스의 바인딩 여부 결정 가능!

2) true/false 값을 가진 객체를 바인딩하는 방법
