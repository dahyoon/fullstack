조건 렌더링 디렉티브

- 웹 앱 개발 시 특정 조건에 따라 화면에 보여줄지 말지 결정해야 될 때 조건 디렉티브 사용 가능
- 조건 렌더링과 관련된 디렉티브: v-show, v-if, v-if-else, v-else 등

v-show

- 화면에 보여줄지 말지 결정하는 디렉티브
- 렌더링 수행하지만 (i.e., HTML 요소는 생성해내지만) 화면에 보여주지 않을 수 있다는 것
- v-show에 부여된 조건이 true일 때만 화면에 보여줌
- v-show에 부여된 조건이 false일 때는 화면에 보여주지 않기 위해 display css 속성이 none으로 부여됨! (개발자 도구 Elements에서 확인 가능)
- 주의: 보안 정보를 다루는 경우 주의 필요 - 화면에 보이지 않지만 값이 HTML 요소를 통해 드러날 수 있기 때문
