v-model 디렉티브

v-model 디렉티브 특징:
- 양방향 데이터 바인딩 지원: 사용자가 UI에서 입력한 값이 내부 데이터에 반영되는 것
- 확인 방법: 개발자 도구에서 console에 vm.name 입력 시 처음에는 빈 문자열 '' 출력하나, 입력란에 문자열을 입력 후 다시 console에서 vm.name 입력 시 사용자가 입력한 값이 반영된 것을 확인

v-model 디렉티브를 이용한 양방향 데이터 바인딩은 checkbox, select, radio button 등에도 적용 가능

주의:
- 다중 선택의 경우는 배열 []을 이용해 데이터 초기화, 단일 선택의 경우 문자열로 데이터 초기화
- 예)
 ...
 data() {
  return {
   hobby: [],
   category: ""
  };
 }
