v-model 디렉티브의 수식어

v-model 디렉티브 특징:
- 양방향 데이터 바인딩 지원: 사용자가 UI에서 입력한 값이 내부 데이터에 반영되는 것

v-model 디렉티브의 수식어 (Modifier):
- 지원하는 수식어 몇가지:
  - lazy: 입력폼에서 다른 요소로 포커스가 이동하는 이벤트 발생 시 입력한 값을 데이터와 동기화함
   - 예시) <input type="text" v-model.lazy="name" />
  - number: 숫자가 입력될 경우 number 타입의 값으로 자동 형변환되어 데이터 옵션 값으로 반영됨 (숫자로 변환 가능한 값은 숫자로 변환, 아니면 문자열로 받아냄)
   - 예시) <input type="text" v-model.number="num" />
  - trim: 문자열의 앞뒤 공백 자동 제거  
   - 예시) <input type="text" v-model.trim="message" />
- Note:
   - 여러 수식어 동시에 부여 가능
     - 예시) v-model.trim.lazy
   - v-model.num
     - (확인 방법: console에서 vm.num 입력)
     - 100 입력 -> number 100 반영
     - 100aaa 입력 -> number 100 반영 (aaa 없어짐)
     - aaa100 입력 -> string aaa100 반영
      - 이유: 앞에서부터 형변환이 가능한 부분만 숫자로 변환함
     - abs 입력 -> string 'abc' 반영
