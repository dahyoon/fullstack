05 이벤트 처리

5.3 이벤트 핸들러 메서드

  • 복잡한 기능은 메서드에 미리 작성해두고, v-on 디렉티브로 참조해서 이벤트 처리 수행

    • (continued 1)

  • 간단한 이벤트 처리 메서드는 인라인으로 템플릿에 작성 가능!

    • (continued 2)

  • 한 줄일 경우 2가지 방법으로 개선 가능
    1. 인라인 이벤트로 처리
      • <input id="a" type="text" :value="name" @input="this.name = $event.target.value">
    2. 인라인 코드로 화살표 함수 등록
      • 주의: 전통적인 함수가 아닌 화살표 함수로 등록해야 this가 컴포넌트 인스턴스를 참조함

입력하신 이름 : {{name}}
