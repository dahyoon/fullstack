5.7 이벤트 수식어

  • .prevent, .stop, .self 외에도 다양한 이벤트 수식어(Event Modifier) 존재

1) once 수식어

  • 한 번만 이벤트 발생시키고 이벤트 연결 해제

(continued 1)

2) 키코드 관련 수식어

  • KeyCode 수식어는 키보드 관련 이벤트 처리 시 사용되는 수식어
  • 키보드의 키를 누를 때 고유의 키코드 (KeyCode) 값을 가질 때만 이벤트 발생시키는 방법 제공

    • (continued 2)

  • keyCode 사용해도 되지만 수식어 사용이 더 간단함 (예, @keyup.enter, @keyup.ctrl.enter, @keyup.ctrl.c, @keyup.tab, etc.)

(continued 3)

3) 마우스 관련 수식어

  • 마우스 버튼 수식어: .left, .right, .middle
- 왼쪽 버튼 : 위로
- 오른쪽 버튼 : 아래로
- CTRL + 왼쪽 버튼 : 처음으로
- CTRL + 오른쪽 버튼 : 마지막으로
