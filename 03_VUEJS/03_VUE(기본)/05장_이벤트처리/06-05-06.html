05 이벤트 처리

5.5 기본 이벤트 (Default Event)

  • 기본 이벤트란: HTML 문서나 요소에 어떤 기능을 실행하도록 이미 정의되어 있는 이벤트
  • 예시
    • <a< 요소 클릭 시 href 특성의 경로로 페이지 이동
    • 브라우저 화면을 마우스 오른쪽 클릭 시 내장 컨텍스트 메뉴 (ContextMenu)가 나타남
    • <form< 요소 내부의 submit 버튼 클릭 시 <form< 요소의 action 특성에 지정된 경로로 method 특성에 지정된 방식으로 전송
    • <input type="text" ... /< 요소에 키보드 누르면 입력한 문자가 텍스트 박스에 나타남
  • 이벤트 실행 중지시키는 방법:
    1. preventDefault() 메서드 사용

      2. (continued 1)

    2. Vue의 이벤트 수식어(Event Modifier)인 prevent 수식어 사용
